# Web Rich Presence

This program links some websites with Discord Rich Presence.
The java program acts like a server and the browser extension connects to it as a client.

Supported websites:
* Soundcloud
* Anime Digital Network
* Crunchyroll
* Wakanim
* Neko-Sama
* FullAnimeVF
* Netflix
* Prime Video

# Firefox add-on
